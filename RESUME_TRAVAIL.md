## 🚀 PRÊT À UTILISER

### Démarrage rapide

**Pour lancer le site complet (frontend + backend):**

```bash
npm run dev:all
```

Ou double-cliquez sur: `START.bat`

Puis accédez à: http://localhost:5173

### Ou séparément

**Frontend uniquement:**
```bash
npm run dev
```
→ http://localhost:5173

**Backend uniquement:**
```bash
npm run server
```
→ http://localhost:5000
